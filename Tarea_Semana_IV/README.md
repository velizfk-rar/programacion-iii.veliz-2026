Sección B

El código crea dinámicamente cuatro nodos dinámicamente, utilizando dos estructuras, una llamada Producto y otra llamada Nodo

El struct de Producto se encarga de crear la estructura que tendrá el producto, en este caso tendrá código, nombre y precio. 

El struct de Nodo se encarga de crear los nodos derivados de Producto, y generando un Nodo llamado siguiente. 

Luego se le pide al usuario ir ingresando los datos para los cuatro nodos, y los va uniendo dinámicamente, se muestra finalmente los productos ingresados
con sus datos incluídos, el total acumulado de la suma de los precios de los productos, y luego se limpia la memoria.
