Descripción

Programa en C++ que permite ingresar números de forma dinámica y los clasifica en estructuras enlazadas según su tipo:

Pila: números impares
Cola: números pares
Lista enlazada: números negativos

Funcionalidades
Ingreso de datos hasta ingresar -1
Inserción automática según clasificación
Operaciones básicas:
Pila: push, pop, peek
Cola: enqueue, dequeue, peek
Lista: insertar, recorrer, mostrar

Validaciones
Control de estructuras vacías
Manejo de entradas inválidas
Uso correcto de memoria dinámica (new/delete)

Extras
Conteo de pares e impares
Ordenamiento de la lista de negativos
Liberación de memoria al finalizar

Cómo ejecutarlo:

1. Compilar g++ codigo_fuente.cpp -o codigo_fuente
2. ./codigo_fuente
